# Curtis Tinkers CMS Starter Recipe

Starter template for the [Curtis Tinkers website](https://curtistinkers.com).

## Recipes applied
- Administrator role (core)
- Recommended Maintenance (core)
- Recommended Performance (core)
- [Curtis Tinkers CMS Blog](https://github.com/curtistinkers/curtistinkers_cms_blog)
- [Curtis Tinkers CMS Forms](https://github.com/curtistinkers/curtistinkers_cms_forms) (soon)
- [Curtis Tinkers CMS Media](https://github.com/curtistinkers/curtistinkers_cms_media)
- [Curtis Tinkers CMS Project](https://github.com/curtistinkers/curtistinkers_cms_project) (soon)
- [Curtis Tinkers CMS Theme Components](https://github.com/curtistinkers/curtistinkers_cms_theme_components)
- [Drupal CMS Admin UI](https://www.drupal.org/project/drupal_cms_admin_ui)
- [Drupal CMS Anti-spam](https://www.drupal.org/project/drupal_cms_anti_spam)
- [Drupal CMS Authentication Tweaks](https://www.drupal.org/project/drupal_cms_authentication)
- [Drupal CMS Basic Privacy](https://www.drupal.org/project/drupal_cms_privacy_basic)
- [Drupal CMS Basic SEO](https://www.drupal.org/project/drupal_cms_seo_basic)
- [Easy Email: Express Mail](https://www.drupal.org/project/easy_email_express)

## Modules and themes installed
- Configuration Manager (core)
- Custom Menu Links (core)
- [Curtis Tinkers Theme](https://github.com/curtistinkers/curtistinkers_theme)
- [CVA (Class Variance Authority)](https://www.drupal.org/project/cva)
- Inline Form Errors (core)
- [Project Browser](https://www.drupal.org/project/project_browser)

## Installation
This project is part of [Drupal CMS](https://drupal.org/project/cms). If you're not using Drupal CMS, you can still use this recipe in your own project. Add it with the following Composer command:

```
composer require drupal/curtistinkers_cms_starter
```
